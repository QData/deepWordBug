Adversarial Playground Text

About

In recent years, Deep Neural Networks have become an essential tool for many machine learning tasks, especially image classification. However, this has given rise to the study of adversarial samples--carefully crafted inputs for a neural network that result in an incorrect classification.

In the spirit of the TensorFlow Playground, we present the Adversarial Playground Text: a web-based visualization tool to assist users in understanding the differences between common adversarial machine learning techniques. This application is based off of a previous project from the QData group: Andrew Norton's Adversarial Playground, however this application focuses on adversarial attacks on text.


About the Attacks


DeepWordBug Algorithm
The core idea of this algorithm is to generate efficient adversarial samples on text input. The algorithm can attack both Char and Word model in a fast and black-box manner.

Notable features of the DeepWordBug are the model, power, transformer algorithm, and scoring algorithm.
